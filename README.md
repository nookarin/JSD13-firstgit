# Gen I Fruit Stand 🍎

An interactive fruit & vegetable explorer built with HTML, CSS, and JavaScript.

## Features

- **50 fruits & vegetables** — each with color, origin, nutrition facts, a fun fact, a health benefit, and 5 dish ideas
- **Search** — filter by name in real time
- **Random pick** — surprise me button
- **Horizontal scroll grid** — swipe through all items
- **Detail card** — photo, stats, benefit, story, nutrition, and clickable menu tags
- **Menu modal** — click any dish to see a photo and simple cooking steps
- **Cute fruit-themed UI** — pastel gradients, rounded cards, floating emoji background

## Files

| File | Description |
|------|-------------|
| `index.html` | Main page — structure, styles, and logic |
| `my-object.js` | Data file with all 50 food items |
| `my-array.js` | Original array example (legacy) |

## Usage

Open `index.html` in any browser. No build step or server required.
